How to create a bookmarklet v1

what is a bookmarklet

a simple way to run JavaScript code to modify browser content via a bookmark button.

How to make a bookmarklet

Create a bookmark button with JavaScript functon. This can be done following way:
  • create an html file containing a 
    
        <a href="JavaScript:(function(){//do whatever you wanna do})()" > link </a>
  • open it in the chrome and drag the link in the upper bar of the browser to make it a bookmark.
